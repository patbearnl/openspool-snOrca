# Security policy

The snOrca OpenSpool project takes security reports seriously.

## Reporting a vulnerability

Preferred: report privately via GitHub Security Advisories:

- https://github.com/patbearnl/openspool-snOrca/security/advisories/new

If you cannot use GitHub Security Advisories, please open a GitHub issue **without** including exploit details and ask for a private contact channel.

## Scope

- This repository (snOrca OpenSpool), including bundled code.
- For third-party dependencies, please also report issues upstream when appropriate.
